# 门禁控制器硬件设计

## 项目简介

本项目是一个基于STM32的门禁控制器硬件设计方案，支持网络通信、多种输入输出接口，适用于门禁管理系统。
该项目是2018年6月设计，8月落地国内某大型技术园区并良好运行，是我刚毕业时独立完成的第二个项目细节不到位的地方可自行优化。

## 硬件规格

### 主控制器

- **型号**: STM32F103RET6
- **内核**: ARM Cortex-M3
- **主频**: 72MHz
- **Flash**: 512KB
- **SRAM**: 64KB
- **封装**: LQFP64

### 网络接口

- **以太网芯片**: W5500
- **接口**: SPI
- **速率**: 10/100Mbps
- **协议支持**: TCP/IP, UDP

### 输出接口

#### 继电器输出

- **数量**: 2路
- **触点类型**: 常开&常闭
- **负载能力**: 250VAC/5A 或 30VDC/5A
- **应用**: 门锁控制、报警输出

#### 蜂鸣器输出

- **数量**: 1路
- **应用**: 报警提示、操作反馈

### 输入接口

#### RS485接口

- **数量**: 3路
- **通信速率**: 最高115200bps
- **隔离**: 光电隔离
- **应用**: 读卡器通信、扩展模块

#### 韦根接口

- **数量**: 2路
- **数据格式**: Data0/Data1
- **应用**: 读卡器输入

#### 开门信号输入

- **数量**: 2路
- **隔离**: 光电隔离
- **应用**: 出门按钮

#### 门磁输入

- **数量**: 2路
- **隔离**: 光电隔离
- **应用**: 门磁状态检测

### 调试接口

- **类型**: UART
- **电平**: TTL（3.3V）
- **备注**: 与RS232共用，焊接RS232后无法使用TTL电平

### 存储与时钟

#### FLASH存储芯片

- **型号**: W25Q64（8MB）
- **接口**: SPI
- **容量**: 8MB/16MB
- **擦写次数**: >100,000次
- **数据保持**: >20年
- **应用**: 固件存储、数据记录

#### EEPROM存储芯片

- **型号**: AT24C02（256B）/ AT24C04（512B）
- **接口**: I2C
- **容量**: 256B/512B

#### 实时时钟芯片

- **型号**: BM8563ESA
- **接口**: I2C
- **备用电源**: 3V纽扣电池
- **闰年补偿**: 自动
- **应用**: 实时时钟、时间记录、定时任务

## 电源规格

- **输入电压**: DC 12V -24V
- **电源保护**: 过压保护、过流保护、反接保护

## 功能特性

- 支持TCP/IP网络通信，可远程控制
- 双路门禁控制，支持双门管理
- 支持多种读卡器接口（韦根、RS485）
- 开门信号实时检测
- 状态输出和报警功能
- 调试接口方便开发和维护
- 光电隔离设计，抗干扰能力强
- 外部FLASH存储，支持大容量数据记录
- EEPROM配置存储，掉电不丢失
- 实时时钟功能，支持时间记录和定时任务

## 开发工具

- 工具: Altium Designer

## 注意事项

1. **调试接口选择**: Debug串口与RS232共用同一物理接口，焊接RS232电平转换芯片后将无法使用TTL电平调试
2. **电源连接**: 上电前请确认电源电压为DC 12V，正负极正确
3. **继电器负载**: 继电器输出连接感性负载时需加装浪涌吸收电路
4. **RS485终端**: RS485总线两端需加装120Ω终端电阻
5. **韦根接线**: 韦根接口线缆长度建议不超过100米
